# Examples

This directory contains example usage scripts for the profanity censor.

## Quick Demo

Run the demo script to see the profanity censor in action:

```bash
./basic_demo.sh
```

## Example Files

- `basic_demo.sh` - Basic usage demo
- `batch_process.sh` - Batch processing example
- `custom_list.sh` - Using custom profanity list
